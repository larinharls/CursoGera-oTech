Valor dos inputs -> .value

Conteudo do HTML de uma tag -> .innerHTML -> processa o código HTML

Conteudo textual de uma tag -> .textContent -> não processa códigos

Manipulação de atributos -> .setAttribute('atributo','valor')

Acessar atributos -> .getAttribute('atributo')

Remover atributos -> .removeAttribute('atributo')

Manipulação de Estilo

Modificação de classe

  • Adicionar .classList.add('Classe')
  • Remover .classList.remove('Classe')
  • Trocar .classList.toggle('Classe')
  • Verificar a classe .classList.contains('Classe')
